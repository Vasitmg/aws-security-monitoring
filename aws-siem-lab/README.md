# AWS SIEM Lab (In Progress)

## Objective

Build a centralized AWS Security Information and Event Management (SIEM) lab 
to aggregate and analyze security events across multiple AWS services.

## Planned Architecture

- AWS CloudTrail
- VPC Flow Logs
- GuardDuty
- Amazon OpenSearch / Athena
- CloudWatch Logs
- SNS Alerting

## Goals

- Centralized log ingestion
- Security event detection
- Threat visualization dashboards
- Automated alerting
- Simulated attack scenarios

## Status

🚧 Currently under development.
